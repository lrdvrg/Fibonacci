## Fibonacci

### Dependencias utilizadas:
Express, chai, chai-http, mocha

## Como instalarlas:
* npm install express
* npm install chai
* npm install chai-http
* npm install mocha

## Para correr el proyecto utilizar el comando: 
node app.js
**Nota: se abrira en el puerto 5255**

## Para correr las pruebas utilizar el comando: 
npm test
