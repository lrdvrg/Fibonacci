# Fibonacci

### Dependencias utilizadas:
Express, chai, chai-http, mocha, cors

### Como instalarlas:
* npm install

>**Nota: Al estar incluidas en el package.json deberian instalarse al ejecutar npm install en caso de que no ejecutar**

* npm install express
* npm install chai
* npm install chai-http
* npm install mocha
* npm install cors
